## 2TB VPS，先搞清楚你要的是哪个"2TB"

这个问题比你想象的重要。

很多人搜索"2TB VPS"的时候，其实混淆了两个完全不同的概念：

**2TB存储空间（Disk Storage）**：指你的服务器硬盘容量是2TB，适合存大量文件、数据库、备份数据、媒体内容。

**2TB月流量（Monthly Bandwidth/Transfer）**：指每月你能传输的数据总量是2TB，适合流量比较高的网站、应用或者内容分发。

一个网站每月访问量在100万PV左右，流量消耗大概在100GB到500GB之间，2TB月流量已经相当宽裕。而如果你在做视频存储、冷备份、或者跑大型数据库，那你需要的是2TB硬盘容量。

明白这个区别之后，我们来看看哪些场景真的用得上。
